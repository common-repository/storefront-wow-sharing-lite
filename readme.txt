== Description ==
A plugin for share the product single page on various social networks
Social networks available: Twitter, Facebook, Google+, Pinterest, LinkedIn, Thumblr, Buffer, StumbleUpon, Reddit, Yahoo, Digg, Line, WhatsApp, Blogger, Email, vk.com, Viber, Skype

== Installation ==

= Minimum Requirements =

* WordPress 4.7 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

Upload the plugin to wp-content/plugins
 Go to WordPress dashboard and  activate the plugin.
Then go to Appearance -> customise -> Storefront Wow Sharing Lite

**Using FTP**

1. Download `storefront-wow-sharing.zip`
2. Extract the `storefront-wow-sharing` directory to your computer
3. Upload the `storefront-wow-sharing` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
6. Go to Appearance -> customise -> Storefront Wow Sharing Lite
